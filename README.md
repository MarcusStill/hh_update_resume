# HH.ru Resume Auto-Lifter 🚀

Автоматический скрипт на Python для периодического поднятия резюме соискателя на сайте hh.ru в обход закрытого официального API. Скрипт эмулирует действия реального пользователя через скрытый браузер, проверяет доступность кнопки обновления и рассчитывает время "умного сна" до следующей разблокировки.

## ✨ Особенности

* **Полная автоматизация**: работает автономно 24/7 в фоновом режиме.
* **Умный расчет времени (Smart Sleep)**: скрипт парсит точное время разблокировки с сайта (например, *"Можно сегодня в 22:36"*) и засыпает ровно до этой минуты, не совершая холостых запросов.
* **Обход ограничений API**: работает через реальный браузер (Playwright Chromium), что защищает аккаунт от блокировок.
* **Фоновый режим**: возможность сборки в скрытый `.exe` файл без консольного окна.
* **Автономное сохранение сессии**: авторизация по СМС/паролю требуется только один раз при первом запуске.
* **Подробное логирование**: все действия и ошибки дублируются в текстовый файл `log.txt`.

## 🛠️ Системные требования и установка

Для работы исходного кода вам понадобятся **Python 3.8+** и менеджер пакетов `pip`.

1. **Клонируйте или создайте папку проекта** и перейдите в нее.
2. **Установите необходимые библиотеки**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Установите встроенный браузер Chromium** для библиотеки Playwright:
   ```bash
   playwright install chromium
   ```

---

## ⚙️ Настройка файлов конфигурации

Рядом со скриптом автоматизации должны находиться два файла.

### 1. `config.ini` (Создать вручную)
Создайте текстовый файл, назовите его `config.ini` и укажите в нем ссылку на ваше резюме:

```ini
[SETTINGS]
# Резервный интервал (в секундах) на случай падения сайта hh.ru или смены его дизайна
fallback_interval_seconds = 14500

[RESUMES]
# Уникальное имя = полная ссылка на ваше резюме
resume_main = https://hh.ru
```

### 2. `auth_state.json` (Генерируется автоматически)
Этот файл содержит куки (`cookies`) вашей учетной записи. Он создается после первого (ручного) входа в систему.

## 🚀 Инструкция по первому запуску (Авторизация)

Так как hh.ru требует ввод СМС-кода или пароля, первый запуск нужно произвести в **видимом режиме браузера**, чтобы перехватить сессию.

1. Измените в коде скрипта строчку `headless=True` на `headless=False` (в блоке инициализации браузера).
2. Запустите скрипт через терминал:
   ```bash
   python hh_updater.py
   ```
3. Откроется окно браузера Chromium на странице логина hh.ru. **Войдите в свой аккаунт соискателя** привычным способом.
4. Как только вы окажетесь в личном кабинете, скрипт автоматически закроет браузер и создаст файл `auth_state.json`.
5. Верните в коде `headless=True`, чтобы браузер больше не открывался на экране.

## 📦 Компиляция в фоновый `.exe` файл

Чтобы скрипт работал незаметно в диспетчере задач Windows и не держал открытым черное окно консоли, его можно упаковать в один исполняемый файл.

1. Установите упаковщик **PyInstaller**:
   ```bash
   pip install pyinstaller
   ```
2. Скомпилируйте проект командой (флаг `--noconsole` скроет окно терминала):
   ```bash
   pyinstaller --noconsole --onefile hh_updater.py
   ```
3. Перейдите в созданную папку `dist`, заберите оттуда файл `hh_updater.exe`.
4. Перенесите `hh_updater.exe` в отдельную постоянную папку и **обязательно положите рядом** ваши файлы `config.ini` и `auth_state.json`.

## 🔄 Автозагрузка Windows (Запуск при включении ПК)

Чтобы бот автоматически запускался в фоне при старте компьютера:
1. Нажмите комбинацию клавиш `Win + R`.
2. Введите команду `shell:startup` и нажмите Enter. Откроется папка автозагрузки.
3. Создайте **ярлык** для вашего файла `script.exe` и перетащите этот ярлык в открывшуюся папку.

## 📝 Чтение логов

Поскольку `.exe` версия работает скрытно, контролировать статус работы бота можно через файл `log.txt`, который автоматически создается в папке с программой.

**Пример идеального лога:**
* `[2026-06-23 19:08:45] Сервис запущен в фоновом режиме с умным расчетом времени.`
* `[2026-06-23 19:08:45] [Проверка] Переход на резюме 'resume_main'...`
* `[2026-06-23 19:08:50] [Отказ] Можно сегодня в 22:36`
* `[2026-06-23 19:08:50] Умный сон: засыпаю на 208 мин. (12489 сек.) до момента разблокировки.`
